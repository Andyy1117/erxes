Color

Our main colors for text, background, border and so on. And you can also costumize the colors with specific functions.

Import#

Import colors from erxes-ui to use our mainly used colors in your project.

import colors from "erxes-ui/lib/styles/colors";

Ecolors#

You can make a color by yourself with hex code using rgb, rgba, darken, lighten functions.

Hex into RGB#

Convert a hex value into RGB with rgb function. Click on the color box to copy the function name.

background: rgb('#673FBD')

Hex into RGBA#

Convert hex to rgb with opacity with rgba function. Click on the color box to copy the function name.

background: rgba('#673FBD', 0.2)

Darken#

Darken the color by darken function. Click on the color box to copy the function name.

background: darken("#673FBD", 30)

Lighten#

Lighten the color by lighten function. Click on the color box to copy the function name.

background: lighten("#673FBD", 30)
